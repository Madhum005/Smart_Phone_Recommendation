# 📱 Smartphone Recommendation Engine Using Multi-Tool Agent

## 📖 Overview

The Smartphone Recommendation Engine is an intelligent system that helps users find the most suitable smartphone based on their preferences, budget, and requirements. The project utilizes a Multi-Tool Agent approach to analyze user inputs, compare smartphone specifications, and generate personalized recommendations.

By combining data processing, decision-making logic, and AI-driven analysis, the system simplifies the smartphone selection process and assists users in making informed purchasing decisions.

## 🚀 Features

* Personalized smartphone recommendations
* Budget-based filtering
* Specification comparison
* Multi-tool agent architecture
* User-friendly recommendation process
* Fast and accurate results

## 🛠️ Technologies Used
